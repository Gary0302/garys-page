01

Core Definitions核心定義

Spiking Neural Network (SNN) 脈衝神經網路
A neural network whose neurons communicate via discrete binary spikes rather than continuous activations. Neurons fire when their membrane potential exceeds a threshold; information is encoded in the timing of these spikes rather than their magnitude.
以離散二值脈衝而非連續激活值進行通訊的神經網路。Neuron 在膜電位超過閾值時發放脈衝，資訊編碼於脈衝的時間而非數值大小。
mem[t] = β · mem[t-1] + input[t] ; spk = 1 if mem > θ else 0
LIF — Leaky Integrate-and-Fire 漏積分發放模型
The standard SNN neuron model. Membrane potential decays at rate β (leaky), integrates input current, and fires when the threshold θ is crossed. A surrogate gradient approximates the Heaviside step function during backpropagation.
標準 SNN neuron 模型。膜電位以速率 β 衰減，積分輸入電流，超過閾值 θ 時發放。Surrogate gradient 在反向傳播時近似 Heaviside 函數。
State Leaky 狀態疊加漏積分
Two cascaded LIF neurons where LIF₁ (β=0.9) feeds its spikes into LIF₂ (β=0.95). This smooths the spike train, spreading neuron firing times across the time axis to increase ordering information density.
兩層串聯的 LIF neuron，LIF₁（β=0.9）的 spike 作為 LIF₂（β=0.95）的輸入。使 spike train 更平滑，讓各 neuron 的發放時間分散在時間軸上。
spk1, mem1 = lif1(cur, mem1) ; spk2, mem2 = lif2(spk1, mem2)
Spike Matrix S 脈衝矩陣
The binary M×T matrix recording all spike events from Pass 1. Row i contains the full spike train of neuron i across T timesteps. This is the uncompressed, raw representation that preserves all ordering information.
Pass 1 輸出的二值 M×T 矩陣，記錄所有脈衝事件。第 i 行是 neuron i 在 T 個時間步內的完整 spike train。
S ∈ {0,1}^(M×T) ; S[i,t] = 1 iff neuron i fires at timestep t
Spike Ordering π 脈衝排列
The permutation of neuron indices ranked by their first spike time. M neurons yield M! possible orderings — the core representational space of this research. No prior work has successfully exploited this space as a primary information carrier.
依據 first spike time 排序的 neuron 索引排列。M 個 neuron 產生 M! 種可能排列——本研究的核心表示空間。目前無任何研究成功利用此空間作為主要資訊載體。
π = argsort([first_spike_time(i) for i in neurons]) ; |Π| = M!
Soft First-Spike Score a 軟性首次脈衝分數
A differentiable approximation of first-spike time. Only the first spike of each neuron contributes, masked by a running-silence product. Temporal weights decay exponentially so earlier spikes receive higher scores. Crucially, this is continuous and its gradient w.r.t. S is non-zero everywhere.
First spike time 的可微分近似。每個 neuron 只有第一個 spike 有貢獻，由 running-silence 乘積遮蔽。時間權重指數衰減，越早的 spike 分數越高。梯度全程非零。
M[i,t] = S[i,t] · ∏_{s<t}(1−S[i,s]) ; a_i = Σ_t M[i,t]·exp(−w·t)
Plackett-Luce Distribution Plackett-Luce 分布
A probability distribution over all M! permutations, parameterized by a score vector a. The probability of a permutation π decomposes as a product of softmax terms over successively smaller sets. The result is a differentiable soft one-hot vector over M! orderings.
由分數向量 a 參數化的 M! 排列概率分布。排列 π 的概率分解為對逐漸縮小的集合的 softmax 乘積。結果是 M! 維的可微分 soft one-hot 向量。
P(π|a) = ∏_{k=1}^{M} exp(a_{π_k}) / Σ_{j≥k} exp(a_{π_j})
Orthogonal Linear Layer 正交線性層
A linear layer constrained to W^T W = I, implemented via Cayley parameterization. It can only rotate the input current space, not scale it. This preserves the relative magnitudes of neuron currents, preventing the model from collapsing ordering into a binary on/off code.
透過 Cayley 參數化約束 W^T W = I 的線性層。只能旋轉輸入電流空間，不能縮放。保留了 neuron 電流的相對大小，防止 ordering 退化為二值碼。
||cur'|| = ||cur|| ; W = (I+S)^{-1}(I−S), S = −S^T (Cayley map)
E-Drop 時間洗牌準確率降幅
The accuracy drop when the time axis of the spike matrix is randomly shuffled at evaluation. A high E-drop confirms the model is genuinely using spike timing information rather than neuron identity or firing rate. E-drop = 0 was the critical diagnostic failure that exposed the binary-code shortcut.
評估時隨機打亂 spike matrix 時間軸後的準確率降幅。高 E-drop 確認模型真正在使用 spike timing 資訊。E-drop = 0 是暴露二值碼捷徑的關鍵診斷失敗。
E-drop = acc(normal) − acc(time_shuffled)

The final POC architecture removes fc_in entirely. Input current is fed directly to LIF₁, preserving the one-to-one correspondence between input dimensions and neuron currents. Any linear layer before the SNN — even an 8×8 identity-like transform — allows the model to rescale currents and collapse ordering into a binary on/off code.
最終 POC 架構完全移除 fc_in。輸入電流直接餵進 LIF₁，保留輸入維度與 neuron 電流的一對一對應關係。任何在 SNN 之前的線性層——即使是 8×8 的小型線性層——都允許模型重新縮放電流，讓 ordering 退化為二值碼。

When an input transformation is needed, an Orthogonal Linear Layer (W^T W = I via Cayley parameterization) preserves current magnitudes while allowing rotation. This is the key architectural insight for future integration with FFN encoders.
需要輸入變換時，正交線性層（Cayley 參數化，W^T W = I）在保留電流大小的同時允許旋轉。這是未來與 FFN 編碼器整合的關鍵架構洞見。

03

Design Evolution設計演進

V1
RPC — Recursive Proportional Cutting
Original design. Spike matrix → argmax → argsort → recursive interval cutting → coordinates. Geometrically sound: similar orderings map to nearby coordinates.
FATAL: argmax + argsort have zero gradient. ∂L/∂W_fc = 0. Pass 1 never learns.
V2
Soft RLS — Soft Recursive Leader Selection
Replaced argmax with exponentially-weighted spike scores a_i. Introduced recursive soft selection via softmax. Gradient flows. But coordinate used neuron index i as value — arbitrary and semantically meaningless.
Softmax collapse when score differences are large. Per-neuron firing rate shortcut not closed. Acc = 0.487.
V3
Plackett-Luce Soft One-Hot
Enumerate all M! permutations. Compute differentiable probability for each via Plackett-Luce. For M>6, use Monte Carlo sampling (K=512) with Gumbel-max trick. Gradient flows through log-probability computation.
Gradient: fc_in norm = 0.014. Acc = 1.000 on toy task. But firing rate still a shortcut.
V4
Direct Current Input + Soft First-Spike Score
Remove fc_in entirely. Input current fed directly to LIF. Replace all-spike score with first-spike-only score using running-silence mask. Add Gaussian noise σ=0.02 to prevent deterministic convergence.
E-drop = 0.744. Ratio = 18.6. First confirmation that SNN genuinely uses spike timing. Binary-code shortcut eliminated by direct input design.
V4+
Orthogonal Layer for fc_in
When an input transformation is needed, constrain W^T W = I via Cayley parameterization. The layer can only rotate, not scale, preserving relative current magnitudes.
E-drop = 0.420 (vs Standard fc: 0.051). Ordering preserved through orthogonal transformation. Path to FFN + SNN co-existence established.

05

Core Findings核心發現

Finding 01 · 發現一
SNN Can Learn Spike OrderingSNN 能學習脈衝排列
With direct current input and soft first-spike scoring, the SNN genuinely exploits temporal ordering as the primary information carrier. Time shuffling causes a large accuracy drop, confirming the mechanism.
在直接電流輸入和軟性首次脈衝分數的設計下，SNN 真正以時間排列作為主要資訊載體。時間打亂導致準確率大幅下降，確認了機制的存在。
E-drop = 0.744 · ratio = 18.6 · acc = 1.000
Finding 02 · 發現二
M! Is the Real Capacity BoundM! 是真實的容量上限
Accuracy and E-drop remain coupled and high until M!/C falls below ~8, then degrade together. The model exclusively uses ordering even as C scales. Capacity degrades gracefully rather than collapsing.
在 M!/C 降到約 8 以下之前，準確率和 E-drop 保持高且耦合。C 增大時模型仍然只使用排列。容量優雅地退化而非崩潰。
acc ≈ E-drop · C=1000 acc=0.733 · C=5000 acc=0.310
Finding 03 · 發現三
Any Linear Layer Kills Ordering任何線性層都會破壞排列
Even an 8×8 standard linear layer before the SNN collapses E-drop from 0.744 to 0.051. The layer rescales currents, creating large magnitude differences that reduce ordering to a binary on/off code. Higher accuracy is not sufficient evidence of ordering use.
即使是 SNN 之前的 8×8 標準線性層，也會讓 E-drop 從 0.744 崩潰至 0.051。線性層重新縮放電流，產生大幅度差異，使排列退化為二值碼。
Standard fc E-drop = 0.051 · baseline = 0.744
Finding 04 · 發現四
Orthogonal Layer Preserves Ordering正交層保留排列
Constraining W^T W = I via Cayley parameterization prevents current rescaling. E-drop remains at 0.420 — substantially above the standard layer's 0.051. Orthogonal accuracy (0.923) even exceeds standard (0.860), showing ordering is beneficial not just preserved.
Cayley 參數化約束 W^T W = I，防止電流重新縮放。E-drop 保持在 0.420，遠高於標準層的 0.051。正交層準確率（0.923）甚至超過標準層（0.860），顯示排列不只被保留，還有益於分類。
Ortho E-drop = 0.420 · orth_err = 0.0000 · acc = 0.923
Finding 05 · 發現五
Recurrent Connections HelpRecurrent 連接有效
LIF₂-level recurrent connections consistently improve accuracy over the base model. The advantage grows with C (C=256: +0.066, C=512: +0.079), suggesting recurrent inter-neuron competition enriches the ordering space under pressure.
LIF₂ 層的 recurrent 連接持續提升準確率。優勢隨 C 增大而增加，顯示 recurrent neuron 間競爭在壓力下豐富了排列空間。
C=512 Rec +0.079 vs Base · advantage grows with C
Finding 06 · 發現六
Gradient Flow Fully Restored梯度流完全恢復
The Plackett-Luce parameterization with soft first-spike scoring restores gradient flow to all layers. The critical fix was replacing argmax/argsort (gradient=0) with differentiable operations throughout the pipeline.
Plackett-Luce 參數化配合軟性首次脈衝分數，恢復了所有層的梯度流。關鍵修正是將 argmax/argsort（梯度=0）替換為全程可微分操作。
fc_in.weight.grad norm = 0.014 · (was: None)
