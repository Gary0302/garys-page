[本地執行](#本地執行) --- ## 問題背景 ### 原始問題 1. **格子座標不準**：`frameOptions.boxes` 為手估比例，與 PNG 相框上的洞位不一致，照片偏位或超出格子。 2. **銀色金屬為橢圓洞**：使用矩形裁切無法貼合橢圓形洞口。 3. **半透明霧化**：部分相框洞為半透明像素（alpha ≈ 130–210），相框疊在照片上會產生白色/灰色霧感。 4. **不透明占位格**：白色 Seoul、天使羽翼等相框洞為不透明白/米色，會完全蓋住底下照片。 ### 曾嘗試但放棄的做法 | 做法 | 問題 | |------|------| | `photoLayer: "above"`（先相框、後照片） | 照片蓋住相框裝飾，只剩照片 | | `globalCompositeOperation: "copy"` | 清除相框像素，背景變空白 | | 僅先照片後相框（不挖洞） | 半透明洞霧化；不透明洞看不到照片 | --- ## 最終解法 ### 統一合成流程（所有相框） ``` ① 鋪背景色（僅銀色金屬需要黑色底） ② 畫入四格照片（依形狀裁切：圓角矩形 or 橢圓） ③ 畫入相框 PNG（格子洞內占位像素已挖空為透明） ``` ### 相框挖洞（`buildFrameWithPunchedHoles`） 在相框 PNG 疊上去之前，掃描每個 `boxes` 區域內的像素，若判定為「洞占位像素」則將 alpha 設為 0： - 已透明（`a < 20`） - 半透明（`a < 245`）→ 消除霧感 - 不透明白/米色占位（`avg > 236` 且色差小）→ 白色 Seoul、天使羽翼 - 不透明粉色占位 → 粉光愛心、粉紅 Music **不透明裝飾**（金屬邊框、愛心、翅膀等）不在上述條件內，保留顯示。 ### 照片裁切 - **矩形格**：離屏 canvas + 圓角矩形 `destination-in` 遮罩 - **橢圓格**（銀色金屬）：離屏 canvas + 橢圓 `destination-in` 遮罩 --- ## 專案結構 ``` gesture-booth/ ├── index.html # 頁面結構 ├── style.css # 樣式 ├── script.js # 主邏輯（手勢偵測 + 合成） ├── frames/ # 相框 PNG（9 款） │ ├── flash-night.png │ ├── black-digital.png │ ├── silver-meta.png │ ├── pixel-star.png │ ├── pink-music.png │ ├── pink-heart.png │ ├── white-seoul.png │ ├── angel-wing.png │ └── full-black.png └── PHOTOBOOTH-FRAME-FIX.md # 本文件 ``` --- ## 相框設定說明 ### 矩形格（多數相框） ```javascript { label: "Flash Night", src: "/frames/flash-night.png", boxes: [ { x: 0.147, y: 0.156, w: 0.332, h: 0.336 }, // 第 1 格（比例座標 0~1） { x: 0.522, y: 0.155, w: 0.332, h: 0.336 }, // 第 2 格 { x: 0.147, y: 0.516, w: 0.332, h: 0.339 }, // 第 3 格 { x: 0.522, y: 0.516, w: 0.332, h: 0.339 } // 第 4 格 ] } ``` ### 橢圓格（銀色金屬） ```javascript { label: "銀色金屬", src: "/frames/silver-meta.png", clipShape: "ellipse", background: "#000000", boxes: [ { cx: 0.289, cy: 0.339, rx: 0.188, ry: 0.123 }, // 中心 + 半徑 { cx: 0.714, cy: 0.339, rx: 0.188, ry: 0.123 }, { cx: 0.289, cy: 0.636, rx: 0.187, ry: 0.123 }, { cx: 0.712, cy: 0.636, rx: 0.189, ry: 0.123 } ] } ``` ### 各相框 boxes 最終值 | 相框 | 格子類型 | 備注 | |------|----------|------| | Flash Night | 2×2 矩形 | 透明洞 | | 黑灰 Y2K | 2×2 矩形 | 透明洞 | | 銀色金屬 | 2×2 橢圓 | 半透明洞 + 黑色背景 | | Pixel Star | 2×2 矩形 | 半透明洞 | | 粉紅 Music | 直排四格 | 不透明粉格 | | 粉光愛心 | 直排四格 | 半透明洞 | | 白色 Seoul | 直排四格 | 不透明白格 | | 天使羽翼 | 直排四格 | 不透明米色格 | | 全黑 Y2K | 直排四格 | 半透明白格 | --- ## 核心函式 | 函式 | 用途 | |------|------| | `drawFinalPhoto()` | 合成入口：背景 → 照片 → 挖洞相框 | | `buildFrameWithPunchedHoles()` | 將相框 PNG 格子洞挖空 | | `isHolePixel()` | 判斷像素是否為洞占位 | | `drawPhotosIntoFrame()` | 依 boxes 畫入四格照片 | | `drawPhotoInRect()` | 圓角矩形裁切 | | `drawPhotoInEllipse()` | 橢圓裁切 | | `getBoxMetrics()` | 統一矩形/橢圓座標換算 | | `drawImageCover()` | 等比填滿（object-fit: cover） | --- ## 完整程式碼 ### index.html ```html AI 手勢四格拍貼機
